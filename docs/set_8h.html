Detailed Description

This file contains definitions for coveo::lazy::set and coveo::lazy::multiset. These lazy-sorted containers mimic the standard C++ containers of the same name, but perform lazy sorting.

These definitions are just typedefs. For actual implementation, see coveo/lazy/detail/lazy_sorted_container.h.
